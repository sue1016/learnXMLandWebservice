# XML 约束的概述和 DTD 的引入

------  

## XML 的约束

#### XML 约束的概述

一个 XML 文档一旦有了约束，那么这个XML文档就**只能使用约束中创建的元素及属性**。       
         
        如果约束中没有创建<a>元素，那么XML文档就不能使用<a>元素！！！  

#### XML 约束类型 

##### DTD 
DTD是平面式文档，DTD文件不是XML文档，通常扩展名为.dtd，它是最早的XML约束。     

##### schema  
schema本身也是XML文档，它比DTD要更加强大，通常命名为.xsd。它是新的xml文档约束 ，用来代替DTD。
