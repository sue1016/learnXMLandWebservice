# XML的基本语法  

## XML元素格式  
类似于HTML  
有始有终的元素里可以有内容，可以有子元素  
也可以是空元素，没有内容，不必有结束标签，就是    

         <tag/>    

元素可以有嵌套   
**只能有一个根元素**  

## 元素中的空白  
对解析没有影响  
## 命名规范  
* 区分大小写  
* 不能以数字开头（下划线可以）  
* 不要用<xml/XML>关键字  
* 不要包含空格
